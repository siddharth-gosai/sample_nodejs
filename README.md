
### Installation
```
npm install
npm install nodemon -g
```

### Debugging
```
export DEBUG=NE:*
```

### Export variables
```
export DEBUG=NE:*
export PORT=3006
export NT_DB_URL=
```

### Run the Application (Development Purpose Only)
```
npm run start
```

### Prepare build
```
npm run build
```


### Run the Application (Production Purpose Only)
```
node dist/serve.js
```
